> Yafai-Skills is distributed through the yafai homebrew tap. To install it, run the following command:

```bash
brew tap yafai-hub/yafai
```
>
> To install the skills, run the following command:

```bash
brew install yafai-skills
```

> Once Installed proceed to write a manifest file.
